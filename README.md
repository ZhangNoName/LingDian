# LingDian

多端业务项目：前台（uni-app）、管理后台（Vue 3）、服务端（Python）。

## 仓库结构

| 目录 | 说明 |
|------|------|
| `uniapp/` | 用户端 / 小程序 / H5 等（uni-app） |
| `frontend-admin/` | 运营管理后台（Vue 3 + Vite + Element Plus，可按需调整） |
| `backend/` | API 与业务逻辑（Python） |
| `docs/` | 架构说明、开发约定与接口文档索引 |

详细说明见 [docs/README.md](docs/README.md)。

## 快速开始

1. 阅读 [docs/06-development-guide.md](docs/06-development-guide.md)。
2. 分别在三个子目录中按各自 README 初始化工程并启动。
